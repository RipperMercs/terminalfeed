VISUAL SPEC v2 · DELIVERY NOTES

This prototype maps 1:1 to the existing TerminalFeed hooks and uses the exact brand tokens from src/index.css. Deliverable format: JSX component + CSS additions that slot into src/App.tsx and src/App.css. No new runtime dependencies. No changes to vite.config.ts, package.json, or worker-additions/.

What's new

  • HEROBtcHero — proposed 2-column span (column-span: all on CSS columns). Uses useBtcPrice for live WS ticks + /api/coingecko/btc-chart for the 24h line. priceFlash / priceFlashRed on tick, pulsing gold dot at last point, 24h range bar with live marker.
  • NEWStatusWall — unified traffic-light wall. Merges useClaudeStatus + useCloudStatus + useDevStatus into one visual. Cells coloured by indicator enum: none→green, minor→amber, major/critical→red, unknown→dim.
  • NEWLiveNowPanel — unified real-time column. Merges ticks from useBtcPrice, edits from useWikipediaLive, new items from useHackerNews, useGithubEvents, useEarthquakes. One concentrated "moving parts" rail.
  • REFRESHMarketsPanel — per-row priceFlash on change, inline sparklines from spark values, tabular-nums alignment. Feeds from useSimStocks.
  • REFRESHCryptoPanel — row-level tick flash, tighter density. Feeds useSimCrypto (CoinCap WS).
  • REFRESHWorldMap — proposed new canvas combining useEarthquakes (lat/lng), useFlightRadar (opensky states), useSpaceLaunches (pad coords). Ripple pings per event.
  • REFRESHWeatherPanel — living scene preserved (sun + palms for LA), adds forecast strip underneath.

Layout invariants respected

  • column-count: 4 main grid with 4px gap — unchanged.
  • Every .panel is height: auto. No min-height, no flex-grow.
  • break-inside: avoid on every panel so masonry doesn't split tiles across columns.
  • Hero BTC uses column-span: all — safely opts out into full-width without breaking masonry.
  • CRT scanlines + phosphor glow overlay kept verbatim on #root::after / #root::before.
  • Ambient body-color shift kept (120s).
  • prefers-reduced-motion disables marquee tracks.

Motion vocabulary (reused, not reinvented)

  • Real-time feeds (useBtcPrice, useSimCrypto, useWikipediaLive): priceFlash/priceFlashRed on tick + livePulse dot.
  • Fast poll (useGasTracker, useAIHub, useFlightRadar): breathing livePulse on dot only.
  • Medium poll (news, reddit, github): no motion; fadeIn on refresh.
  • Slow/daily (useNasaApod, useHumansInSpace, useRecipe): no motion. "Card of the day" stillness.
