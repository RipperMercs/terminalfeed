>_ About TerminalFeed

TerminalFeed.io is a free, real-time command center dashboard. Think of it as a Bloomberg Terminal for the rest of us — live crypto prices, stock market data, breaking news, earthquake monitoring, Wikipedia edits streaming in real-time, and 30+ more data feeds. All on one dark, terminal-themed page.

No login. No paywall. No tracking. Just data.

The site was built for people who want a second monitor dashboard that actually stays useful. Traders watching BTC and stocks. Developers monitoring GitHub trending and service status. Nerds who want to see earthquakes and space launches alongside their market data. Everyone who wants the internet's most interesting data streams in one place.

Built By

TerminalFeed is built by Ripper — an independent developer who wanted a better second monitor. The site is a personal project, not a corporation. It's free because the best tools should be accessible to everyone.

The Tech

Built with React + Vite + TypeScript. Deployed on Cloudflare Pages. Uses 10+ WebSocket and Server-Sent Event connections for true real-time data. Self-healing API system automatically hides panels when feeds fail and restores them when they recover.

No backend required. Every data feed runs client-side with localStorage caching and static fallbacks. The site loads in under 2 seconds and stays stable for 8+ hours.
