Developer Tools

Thirty single-page browser tools that run entirely on your machine. No analytics on these pages, no requests to a backend on submit, no telemetry. Bookmark them, use them offline, paste them in incognito tabs. Every tool below loads in under 200ms and respects your time. Full index also at /tools.

Encoders & Decoders
Base64
Encode and decode strings or files. Round-trip safe, supports binary upload, copy with one click.
EncodingFiles
Hex Converter
Two-way between bytes, integers, ASCII, and color hex. Useful for debugging binary protocols and CSS.
EncodingColor
URL Encoder
Percent-encode and decode query strings, paths, and fragments. Handles unicode, plus-vs-space variants.
WebEncoding
HTML Entities
Encode and decode &amp; &lt; &gt; and 250+ named/numeric entities. For XSS-safe output and email templates.
WebEncoding
JWT Decoder
Decode JWTs, inspect header/payload/signature, check expiry. Decoder runs locally, never leaves your browser.
AuthSecurity

Markdown Preview
Live render, GitHub-flavored, copy HTML output. Tables, fenced code, task lists all supported.
WebWriting
