Public REST API

Thirty-plus free REST endpoints across crypto, markets, news, status, and world events. No API key, no signup, CORS open on every endpoint, stale-cache fallback so you never get a 5xx. The full machine-readable contract is at /openapi.json. Agent discovery file at /llms.txt. Tool definitions for OpenAI / Anthropic / raw at /api/llm-tools.

>_ Quick start

One curl command, zero auth, returns a composed world snapshot pulling from five upstream sources. This is the endpoint AI agents tend to call first.

# Compose a world snapshot for an LLM context window
curl https://terminalfeed.io/api/briefing | jq .
