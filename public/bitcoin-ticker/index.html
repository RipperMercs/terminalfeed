Real-Time Bitcoin Price Sources

The Bitcoin ticker on this page pulls from a primary and a fallback source. The primary is Binance, streaming trade tick data via WebSocket with zero polling interval. When the WebSocket connection fails, which can happen on corporate networks, with aggressive ad blockers that block Binance, or during library upgrades, the ticker falls back to CoinCap, polling every few seconds.

This dual source architecture means TerminalFeed's Bitcoin ticker keeps working when single source tickers go stale. You can also pull the latest price directly from the public API at /api/btc-price, which returns JSON with CORS enabled and no authentication required.

Frequently Asked Questions

Is the Bitcoin ticker free?

Yes. No signup, no subscription, no rate limit. Anyone can view the ticker, embed it, or pull the price from the public API.

How often does the Bitcoin price update?

On desktop the main dashboard ticker updates every second via Binance WebSocket. On mobile it throttles to every three seconds to preserve battery life. Most competing free tickers update every 5 to 60 seconds.

What if Binance goes down?

The ticker automatically falls back to CoinCap. The switchover is silent. Users see no interruption.

Can I embed the Bitcoin ticker on my site?

Yes. Embed code is available at terminalfeed.io/widgets. The embeddable ticker is sandboxed and ad free.

Does the ticker have an API?

Yes. The endpoint is https://terminalfeed.io/api/btc-price. It returns JSON, is CORS enabled, and requires no authentication. Full API docs at /developers.
