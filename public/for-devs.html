TerminalFeed for Developers

No Tracking No Signup No Popups Free Public API Hosted MCP Server USDC on Base

A focused developer surface inside TerminalFeed.io. Twenty-four browser-based dev tools that run locally with no server round-trip. A public REST API across thirty real-time data sources, no API key required. A hosted Model Context Protocol server with twenty-seven tools. >_ Hosted MCP Server

Twenty-seven tools exposed via the Model Context Protocol. Drop the snippet below into Claude Desktop or Claude Code config, no auth required for the free tier. Pass an Authorization header to access the agent-payable premium tier.

{
  "mcpServers": {
    "terminalfeed": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://terminalfeed.io/api/mcp"
      ]
    }
  }
}

MCP discovery endpoint: /api/mcp. Tool catalog in OpenAI, Anthropic, or raw format: /api/llm-tools. Already listed on the Official MCP Registry, Smithery, and the Glama AI catalog.

>_ Agent-Payable Premium Tier

For autonomous agents that need composed, normalized, multi-source data, the premium tier (/api/pro/*) accepts USDC micropayments on Base. No OAuth, no signup, no credit card. The agent buys credits, gets a bearer token, and starts calling. Credits are federated with TensorFeed.ai: one purchase, two domains.

Pricing: 1 credit = $0.02. A typical 2-credit call costs four cents. Six more endpoints at /developers/agent-payments. Working integration walkthrough: "How AI Agents Pay for Real-Time Data on TerminalFeed".
