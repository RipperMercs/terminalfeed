>_ Hosted MCP Server

27 tools · Free tier no auth · Premium tier USDC bearer

Twenty-seven tools exposed via the Model Context Protocol. Drop the snippet below into Claude Desktop or Claude Code config, no auth required for the free tier. Pass an Authorization header to access the agent-payable premium tier.

{
  "mcpServers": {
    "terminalfeed": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://terminalfeed.io/api/mcp"
      ]
    }
  }
}

MCP discovery endpoint: /api/mcp. Tool catalog in OpenAI, Anthropic, or raw format: /api/llm-tools. Already listed on the Official MCP Registry, Smithery, and the Glama AI catalog.

>_ Code Recipes

Paste-and-go snippets · JS, Python, MCP, agent

Tested recipes for the most common integration patterns. Drop into a project and they work.

Live BTC ticker in vanilla JS, no dependencies
Polls /api/btc-price every 30 seconds. Renders price + 24h change.
async function tick() {
  const r = await fetch('https://terminalfeed.io/api/btc-price');
  const d = await r.json();
  document.querySelector('#btc').textContent =
    '$' + d.data.price.toLocaleString() +
    ' (' + d.data.change24h.toFixed(2) + '%)';
}
tick();
setInterval(tick, 30000);
One-call world briefing in Python
Pulls the composed /api/briefing snapshot for an LLM context window.
import requests, json
r = requests.get('https://terminalfeed.io/api/briefing')
briefing = r.json()
# briefing['data'] has crypto, fear_greed, earthquakes, hn, astros
print(json.dumps(briefing, indent=2))
