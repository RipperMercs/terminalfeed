›_ CRYPTOCURRENCY REFERENCE

Every major cryptocurrency on its own page. 29 utility-bearing coins covered across Layer 1 chains, L2s, DeFi, stablecoins, AI/compute, privacy, and infrastructure. Each entry has live price data, the actual technical story, real-world use cases, honest tradeoffs, and FAQ.

Editorial note: this list deliberately excludes memecoins and tokens with no utility narrative. Per our editorial policy, we cover crypto as data, not as gambling.

L1

ADA
Cardano
A research-driven proof-of-stake blockchain emphasizing formal verification, peer-reviewed protocol design, and academic rigor.
APT
Aptos
A Layer 1 blockchain built by former Meta engineers, using the Move programming language originally developed for the Diem project.
ATOM
Cosmos
The native token of the Cosmos Hub, the central blockchain of an interoperable ecosystem of independent chains connected via IBC.
AVAX
Avalanche
A high-throughput Layer 1 blockchain with sub-second finality, built around three interoperable chains and a subnet model for custom networks.
BTC
Bitcoin
The first decentralized cryptocurrency, running on a proof-of-work blockchain since January 2009. The reference asset for the entire crypto market.
DOT
Polkadot
A multi-chain protocol where independent blockchains (parachains) share security and interoperate through a central relay chain.
ETH
Ethereum
The dominant smart-contract platform, settling most of the activity in DeFi, NFTs, stablecoins, and Layer 2 rollups.
INJ
Injective
A Cosmos-based Layer 1 optimized for financial applications, with native order-book infrastructure and cross-chain interoperability.
NEAR
NEAR Protocol
A sharded Layer 1 blockchain optimized for usability, with human-readable account names and built-in account abstraction.
SEI
Sei Network
A high-throughput Layer 1 blockchain optimized for trading applications, with parallelized execution and a native order book matching engine.
SOL
Solana
A high-throughput Layer 1 blockchain optimized for low fees and fast finality, popular for DeFi, payments, and consumer applications.
SUI
Sui
An object-oriented Layer 1 blockchain using a parallelized execution model, built by former Meta Diem engineers using the Move language.

L2

ARB
Arbitrum
The largest Ethereum Layer 2 by total value locked, an optimistic rollup that runs the Ethereum Virtual Machine with much lower fees.
OP
Optimism
An Ethereum Layer 2 using the OP Stack, the open-source rollup framework powering an interoperable "Superchain" of major L2s.
POL
Polygon
A multi-chain Ethereum scaling ecosystem including the Polygon PoS chain, zkEVM, and the AggLayer cross-chain settlement layer.

DeFi

AAVE
Aave
A leading decentralized lending protocol where users deposit collateral to borrow assets, with floating interest rates set by utilization curves.
LINK
Chainlink
The dominant decentralized oracle network, providing on-chain price feeds, randomness, and cross-chain messaging to most of DeFi.
MKR
Maker
The governance and recapitalization token of MakerDAO, the protocol behind DAI, the original decentralized stablecoin.
UNI
Uniswap
The governance token of Uniswap, the largest decentralized exchange on Ethereum and the original AMM that defined DeFi.

Stablecoin

DAI
Dai
The original decentralized over-collateralized stablecoin, soft-pegged to the US dollar and issued by the MakerDAO protocol.
USDC
USD Coin
A US-dollar-pegged stablecoin issued by Circle, fully backed by short-term Treasuries and cash, favored by institutions and DeFi protocols.
USDT
Tether
The largest US-dollar-pegged stablecoin by circulating supply, used for trading, settlement, and dollar access in markets without USD banking.

AI

RENDER
Render Network
A decentralized GPU rendering network where artists pay node operators to render 3D scenes, generative AI, and video production at scale.
TAO
Bittensor
A decentralized machine intelligence network where subnets compete to provide AI services like inference, search, and image generation.

Privacy

XMR
Monero
The leading privacy-focused cryptocurrency, using ring signatures, stealth addresses, and confidential transactions to obscure sender, recipient, and amount by default.

Infra

FIL
Filecoin
A decentralized storage network where node operators provide hard-disk capacity for hosting files, paid in FIL.
TIA
Celestia
The first modular blockchain providing data availability as a service to other chains, enabling cheap rollup deployment.

Exchange

BNB
BNB
The native token of the BNB Chain ecosystem, originally launched as the Binance exchange utility token, now powers a major EVM-compatible blockchain.

Payment

XRP
Ripple
A payment-focused digital asset designed for fast, low-cost cross-border settlement, with a unique consensus model and a US regulatory history.
