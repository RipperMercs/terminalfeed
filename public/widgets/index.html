Want to build your own?

Every widget pulls from a public TerminalFeed API endpoint. The full catalog is at /api, with LLM-ready tool definitions at /api/llm-tools and an MCP server at /api/mcp for AI agents. Most data feeds (BTC price, fear & greed, crypto markets, status, earthquakes, HN, weather, predictions) are free with no signup.

Hand-crafted dashboards or trading bots can use the same APIs the widgets do. For aggregated, normalized, paid endpoints (correlations, whale watch, exchange flows, deep crypto profiles), see /for-devs/api. Bug reports and widget requests welcome at hello@terminalfeed.io.
