Common Network Ports Reference

What Are Network Ports?

A network port is a numbered endpoint for communication. When a computer sends or receives data over a network, it uses port numbers to route traffic to the correct application. Ports range from 0 to 65535. Ports 0-1023 are "well-known" ports assigned to common services by IANA.

TCP vs UDP

TCP (Transmission Control Protocol) provides reliable, ordered delivery of data. Used for web, email, file transfer, and most applications. UDP (User Datagram Protocol) is faster but unreliable — used for DNS, streaming, gaming, and VoIP.
