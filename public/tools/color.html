Color Converter — HEX, RGB, HSL & Contrast Checker

What Is a Color Converter?

A color converter translates color values between different formats used in web development and design. The most common formats are HEX (e.g., #5DCAA5), RGB (e.g., rgb(93, 202, 165)), and HSL (e.g., hsl(160, 49%, 58%)). Each format represents the same color in a different way.

WCAG Contrast Ratios

The Web Content Accessibility Guidelines (WCAG) define minimum contrast ratios between text and background colors. AA requires at least 4.5:1 for normal text and 3:1 for large text. AAA requires 7:1 for normal text and 4.5:1 for large text. Meeting these ratios ensures your content is readable by users with visual impairments.

How to Use This Tool

Enter any color value (HEX, RGB, or HSL) in the input field, or use the color picker. All formats update instantly. Set a background color to check WCAG contrast ratios. Click any swatch in the palette section to set it as the active color.
