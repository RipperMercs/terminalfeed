    What Is a Cron Expression?

    A cron expression is a string of five (or six) fields that defines a recurring schedule. Originally from the Unix cron daemon, this syntax is now used in Linux crontabs, CI/CD pipelines, cloud schedulers (AWS CloudWatch, Google Cloud Scheduler), Kubernetes CronJobs, GitHub Actions, and task runners everywhere. Each field represents a time unit: minute hour day-of-month month day-of-week.

    Cron Syntax Reference

    FieldAllowed ValuesSpecial Characters
    Minute0–59* , - /
    Hour0–23* , - /
    Day of Month1–31* , - /
    Month1–12* , - /
    Day of Week0–7 (0 & 7 = Sun)* , - /

    Special Characters

    * = every value. , = list (e.g., 1,3,5). - = range (e.g., 1-5). / = step (e.g., */5 = every 5 units).
