HTML Entity Encoder & Decoder

What Are HTML Entities?

HTML entities are special codes used to display reserved characters in HTML. Characters like <, >, &, and " have special meaning in HTML, so they must be encoded as entities (&lt;, &gt;, &amp;, &quot;) when you want to display them as literal text.

Why Encode HTML?

Encoding HTML entities prevents XSS (Cross-Site Scripting) attacks by ensuring user input is rendered as text, not executable HTML or JavaScript. It's also necessary when displaying code samples, embedding user-generated content, or working with special characters like copyright symbols, em dashes, and mathematical operators.
