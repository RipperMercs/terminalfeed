About the TerminalFeed API Playground

The API Playground is an interactive tool for developers to test all TerminalFeed endpoints without authentication. Every endpoint returns live JSON data from our real-time feeds covering cryptocurrency, stocks, weather, earthquakes, prediction markets, and more.

How to Use

Click the "Try it" button on any endpoint card to fetch live data. You'll see the JSON response, response time in milliseconds, and options to copy the response or generate code snippets. The Fetch All button runs all 19 endpoints simultaneously for load testing.

API Endpoints

Available endpoints include briefing (world snapshot), BTC price, stocks, crypto movers, fear and greed index, earthquakes, Polymarket predictions, HackerNews, service status, cyber threats, GitHub trending, forex rates, ISS crew, disaster alerts, space launches, economic data, Steam games, weather, and API call statistics.

Code Generation

Select an endpoint from the code generator dropdown to see copy-paste code examples in cURL, JavaScript (fetch), and Python (requests). Perfect for integrating TerminalFeed data into your own projects.

Developer Resources

Visit the full API documentation for detailed endpoint specs, parameters, and response schemas. Check out the blog for tutorials and best practices. For questions, reach out to hello@terminalfeed.io.
