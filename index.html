TerminalFeed: Free Real-Time Dashboard for Crypto, Stocks, News and More

No Tracking No Popups No Signup Free Public API Built for Devs and Agents

TerminalFeed is a free, real-time data dashboard built for developers, traders, and AI agents. It includes a free live Bitcoin ticker with sub-second price updates, and pulls from 30+ additional data sources on a single dark, terminal-themed page. Think of it as a Bloomberg Terminal without the $25,000 price tag. No accounts, no paywalls, no tracking cookies, no Google Analytics, no Facebook pixels, no popups, no autoplay.

Live Bitcoin Ticker and Market Data

The real-time Bitcoin ticker at the top of the page streams the live BTC price via Binance WebSocket with one-second updates on desktop and automatic fallback to CoinCap when the primary stream drops. You can also pull the latest price directly from the free public API at /api/btc-price. Beyond the ticker, monitor the top cryptocurrency movers sorted by 24 hour change, BTC network stats including block height, mempool size, hashrate, and transaction fees. Check the Crypto Fear and Greed Index, global crypto market cap, BTC dominance, and daily trading volume. Watch large whale transactions on the Bitcoin network as they happen. Follow stock market movers, forex currency pairs, commodity prices, and global market hours across exchanges worldwide.

News, Tech, and World Monitoring

TerminalFeed aggregates tech and AI headlines from Hacker News, Reddit, and curated RSS feeds. Stream Wikipedia edits in real time. Monitor earthquake activity from USGS with live seismic event data. Track global disaster alerts via GDACS, upcoming space launches, cybersecurity threats from URLhaus and ThreatFox, prediction market odds from Polymarket, service status for GitHub, Cloudflare, Discord, and OpenAI, and more. Every panel updates automatically so the page always reflects what is happening right now.

Free Developer Tools and Public API

TerminalFeed includes a suite of free, client-side developer tools: JSON formatter, Base64 encoder and decoder, UUID generator, Unix timestamp converter, JWT decoder, regex tester, and more. All tools run in the browser with no signup required. For programmatic access, every data feed is available through the free TerminalFeed API at terminalfeed.io/api with endpoints for Bitcoin price, stocks, crypto movers, earthquakes, weather, news, and a one-call world briefing used by AI agents for real-time context.

Our Network

TerminalFeed is part of a network of independent data platforms. Visit TensorFeed.ai for AI news aggregation and intelligence tracking.

Agent Fair-Trade Certified

TerminalFeed is agent fair-trade certified: code-enforced no-charge guarantees on 5xx, breaker, schema fail, and stale data; Ed25519-signed receipts on every paid call; a public on-chain payment rail (USDC on Base). Built with Claude. A bearer token minted on terminalfeed.io or tensorfeed.ai works on both. Read the standard at /agent-fair-trade or fetch the manifest at /.well-known/agent-fair-trade.json.
